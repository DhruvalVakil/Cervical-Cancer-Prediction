# Cervical-Cancer-Prediction

## 📌 Project Overview:
Cervical cancer is a significant health challenge globally, yet early detection can vastly improve outcomes. This project focuses on developing a machine learning model to predict cervical cancer risk using biopsy-based tabular medical data. By identifying key indicators from historical patient data, this model aims to assist healthcare professionals in early risk assessment and data-driven decision-making.

## 🔑 Key Objectives & Achievements:
* **Built ML models to predict cervical cancer risk with 95% accuracy** using a biopsy-based tabular dataset.
* **Evaluated predictive performance using medical-standard metrics** including precision, recall, F1-score, and AUC-ROC to ensure robustness in a clinical context.
* **Identified significant features** within the medical dataset contributing to cancer risk.
